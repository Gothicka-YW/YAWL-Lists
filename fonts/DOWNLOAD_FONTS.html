📥 Setup Nunito Font for YoInventory Suite

Step 1: Download Nunito from Google Fonts

Click this link to download Nunito:

🔗 Download Nunito Font (Google Fonts Direct Link)

Alternative: Visit fonts.google.com/specimen/Nunito, click "Get font" → "Download all"

Step 2: Extract the ZIP

Open the downloaded Nunito.zip file and extract it to a folder.

Step 3: Copy Required Fonts

From the extracted folder, navigate to Nunito/static/ and copy these 3 files to your extension's fonts/ directory:

  • Nunito-Regular.ttf
  • Nunito-SemiBold.ttf
  • Nunito-Bold.ttf

Full path: YoInventory_Suite/fonts/

✅ You're Done!

Once the 3 font files are in the fonts/ folder:

  1. Reload your Chrome extension
  2. Open YoInventory Suite
  3. Export a PNG to see the new rounded, friendly Nunito font!
Optional: Convert to WOFF2 (smaller file size)

If you want smaller file sizes, you can convert the TTF files to WOFF2:

  1. Go to cloudconvert.com/ttf-to-woff2
  2. Upload each TTF file
  3. Download the converted WOFF2 files
  4. Place them in fonts/ alongside the TTF files (both formats are supported)
