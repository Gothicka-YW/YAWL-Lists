# Fonts for YoInventory Suite

## Nunito Font (Rounded & Friendly)

**Download from Google Fonts:**
1. Go to https://fonts.google.com/specimen/Nunito
2. Click "Get font" → "Get font" → "Download all"
3. Extract the ZIP file
4. Copy these files to this `fonts/` directory:
   - `Nunito/static/Nunito-Regular.ttf` (rename to `Nunito-Regular.ttf`)
   - `Nunito/static/Nunito-Bold.ttf` (rename to `Nunito-Bold.ttf`)
   - `Nunito/static/Nunito-SemiBold.ttf` (rename to `Nunito-SemiBold.ttf`)

**Alternative Quick Method:**
If you prefer woff2 (smaller file size), convert the TTF files using:
- https://cloudconvert.com/ttf-to-woff2

## Required Files:
- ✅ `Nunito-Regular.ttf` (or .woff2)
- ✅ `Nunito-SemiBold.ttf` (or .woff2)
- ✅ `Nunito-Bold.ttf` (or .woff2)

After adding the font files, the extension will automatically use Nunito for PNG exports.
